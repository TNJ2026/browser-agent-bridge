🔌 申请的系统权限 / System Permissions Requested:

  • debugger (调试器): 通过开发者工具协议 (CDP) 模拟鼠标点击、键盘输入、页面滚动、悬停、截图，以及记录控制台与网络日志。
    Simulates mouse clicks, keyboard input, page scrolling, hovering, capturing screenshots, and recording console/network logs via CDP.
  • activeTab / scripting (活动标签页/脚本注入): 获取活动网页的文本和结构信息，并向页面注入辅助工具脚本。
    Accesses active webpage text/structure and injects helper scripts.
  • nativeMessaging (本地消息通信): 连接并启动本地运行的 Python 宿主后台程序。
    Connects to and launches the locally running Python host backend.
  • declarativeNetRequest (网络请求拦截): 实现 CSP (内容安全策略) 的绕过开关，方便本地 Agent 操控网页。
    Intercepts and strips CSP headers to facilitate page script execution by the Agent.
  • storage (本地存储): 存储插件配置项（例如您的此项同意状态及 CSP 绕过设置）。
    Stores configuration settings (such as agreement status and CSP bypass settings).
  • tabs / tabGroups (标签及群组): 创建、激活、编组、关闭浏览器标签页以管理工作区。
    Creates, activates, groups, and closes browser tabs to manage workspaces.
  • downloads (下载管理): 列出并管理浏览器下载任务。
    Lists and manages browser download tasks.

🔒 可能会读取的隐私和数据信息 / Private Information & Data Collected:

  • 本地浏览器历史记录 / Local Browser History：通过本地 Python 宿主读取 Chrome/Edge 浏览器历史数据库（仅在本地响应 Agent 查询，绝不外发）。
    Reads Chrome/Edge browser history databases locally (only queries locally for your Agent, never sent externally).
  • 本地浏览器书签 / Local Browser Bookmarks：读取本地书签文件以检索快捷方式。
    Reads local bookmarks files to retrieve shortcuts.
  • 页面会话数据与快照 / Page Session Data & Snapshots：获取当前打开页面的完整 URL、标题、DOM 树、可读文本，以及网页视觉截图。
    Accesses current page URLs, titles, DOM trees, readable text, and screenshots.
  • 页面操作记录 / Page Interaction Recording：在录制模式下记录您对页面的交互行为。
    Records your page interaction activities when recording mode is active.

⚠️ 绕过 CSP 的风险提示 / Risks of Bypassing CSP:

本插件提供了可选的“Bypass CSP”选项，开启后会拦截并剥离网页的 Content Security Policy 响应头。这能让您的 Agent 顺利在任意网页注入和运行自定义脚本，但它会降低浏览器对跨站脚本 (XSS) 攻击的防御能力。在开启此功能期间，请避免浏览不受信任的网页，并确保仅运行来自可信来源的 Agent 任务。
This extension provides an optional "Bypass CSP" feature which intercepts and strips Content Security Policy response headers. While this allows your Agent to run custom scripts on any webpage, it lowers the browser's defense-in-depth security against Cross-Site Scripting (XSS) attacks. During the period this feature is active, please avoid browsing untrusted webpages and ensure you only execute Agent tasks from trusted sources.

Please ensure that the local Agent you run is also secure and trusted.
