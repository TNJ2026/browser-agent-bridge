免责声明与隐私说明

欢迎使用 Local Browser Agent Bridge。在启用本插件前，请仔细阅读以下关于本插件所需申请的系统权限以及可能会读取的隐私和数据信息的声明：

🔌 申请的系统权限：

  • debugger (调试器): 通过开发者工具协议 (CDP) 模拟鼠标点击、键盘输入、页面滚动、元素悬停、抓取页面截图，以及记录控制台与网络请求日志。
  • activeTab / scripting (活动标签页/脚本注入): 获取活动网页的文本和结构信息，并向页面注入辅助工具脚本。
  • nativeMessaging (本地消息通信): 连接并启动本地运行的 Python 宿主后台程序（进行进程间通信）。
  • declarativeNetRequest (网络请求拦截): 用于实现 CSP (内容安全策略) 的绕过开关，方便本地 Agent 进行自由的网页脚本操控。
  • storage (本地存储): 存储插件配置项（例如您的此项同意状态及 CSP 绕过设置）。
  • tabs / tabGroups (标签及群组): 创建、激活、编组、关闭浏览器标签页以管理任务工作区。
  • downloads (下载管理): 列出并管理浏览器下载任务。

🔒 可能会读取的隐私和数据信息：

  • 本地浏览器历史记录：通过本地 Python 宿主直接读取 Chrome/Edge 浏览器历史数据库（仅在本地响应 Agent 查询请求，绝不外发）。
  • 本地浏览器书签：读取本地书签文件以检索快捷方式。
  • 页面会话数据与快照：获取当前打开页面的完整 URL、标题、DOM 树、可读文本，以及页面视觉渲染截图。
  • 页面操作记录：在录制模式下记录您对页面的交互行为。

⚠️ 安全承诺与提示：

本插件和本地宿主服务完全运行在您本地的系统环境中，不包含任何远程云端服务或数据上报代码。所有敏感信息仅通过本地回环地址端口（127.0.0.1）提供给您自己运行的本地 Agent 使用，绝不会上传给任何第三方。请确保您运行的本地 Agent 也是安全可信的。

您必须同意免责声明才能开始使用本插件。
